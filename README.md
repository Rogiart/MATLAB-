# MATLAB Projects

## 概要
大学の授業で作成したMATLABプログラムをまとめたリポジトリです。

## 使用技術
- MATLAB

## 収録内容

### 画像情報処理
- 画像の二値化
- フィルタ処理
- エッジ検出

### 音生成・音声処理
- 正弦波生成
- 音声波形解析
- 周波数解析
- フーリエ変換

### 機械学習
- データ前処理
- モデル学習
- 分類・予測
- 精度評価

## 工夫した点
処理結果を比較しやすいようにグラフや画像による可視化を行った。また、パラメータを変更しながら結果の違いを分析できるようにした。

## 学んだこと
画像処理、音声処理、機械学習などの情報処理技術について学び、MATLABを用いたデータ分析や可視化の基礎を習得した。
